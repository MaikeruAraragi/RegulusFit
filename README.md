# RegulusFit

**RegulusFit** es el proyecto web inicial de una marca deportiva centrada en entrenamiento en gimnasio físico en Colombia, con visión de crecer hacia una comunidad fitness activa y venta de merchandising en el futuro.

## Misión

Ofrecer un espacio de entrenamiento auténtico donde la disciplina, la técnica y la motivación se unen para que cada persona entrene como un campeón.
